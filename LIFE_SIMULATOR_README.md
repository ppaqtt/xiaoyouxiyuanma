# 人生模拟器 (Life Simulator)

一个使用 pygame 开发的中文人生模拟游戏，从出生到死亡，体验完整的人生旅程！

## 游戏特点

- **完整的人生阶段**：6个不同的人生阶段（婴儿、儿童、青少年、成年、中年、老年）
- **选择系统**：每个阶段有不同的生活选择，影响人生轨迹
- **属性系统**：4个核心属性（健康、幸福、财富、学历）
- **随机事件**：30%的概率触发随机事件，增加游戏乐趣
- **音效系统**：使用 bytearray 生成音效，无需外部音频文件
- **人生总结**：游戏结束时给出完整的人生评价

## 如何运行

确保你已安装 pygame：
```bash
pip3 install pygame
```

然后运行游戏：
```bash
python3 life_simulator.py
```

## 游戏玩法

1. 游戏从婴儿期开始
2. 每个年龄你都会面临不同的选择
3. 点击按钮做出你的选择
4. 你的选择会影响四个属性的变化
5. 每年年龄增长1岁
6. 当健康值降为0时，人生结束
7. 查看人生总结，然后可以选择重新开始

## 人生阶段

| 阶段 | 年龄范围 | 特点 |
|------|----------|------|
| 婴儿 | 0-2岁 | 简单选择，基础成长 |
| 儿童 | 3-11岁 | 学习与玩耍的平衡 |
| 青少年 | 12-17岁 | 关键抉择期 |
| 成年 | 18-39岁 | 事业与家庭 |
| 中年 | 40-59岁 | 稳定与发展 |
| 老年 | 60岁+ | 安享晚年 |

## 属性说明

- **健康**：绿色条，决定你能活多久
- **幸福**：黄色条，生活满意度
- **财富**：蓝色条，经济状况
- **学历**：橙色条，教育程度

## 技术特点

- 纯 pygame 实现，无需其他依赖
- 音效使用正弦波合成，无需外部文件
- 响应式 UI 设计
- 支持重新开始功能
