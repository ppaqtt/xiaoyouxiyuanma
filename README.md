# 🎮 Python 小游戏合集

欢迎来到 Python 小游戏合集！这里包含了 **149个** 精美的 Python 小游戏，涵盖了各种类型和难度。

## 📁 文件说明

### 启动器
- **game_launcher.py** - 游戏启动器 v1.0（标准版）
- **game_launcher_v2.py** - 游戏启动器 v2.0（改进版，推荐使用）

## 🚀 快速开始

### 1. 安装依赖
```bash
pip install pygame
```

### 2. 使用游戏启动器（推荐）
```bash
python game_launcher_v2.py
```

### 3. 直接运行游戏
```bash
python snake_game.py          # 贪吃蛇
python game_2048.py           # 2048
python tetris.py              # 俄罗斯方块
```

## 🎨 游戏启动器功能

### 界面特点
- 🎨 美观的渐变界面
- 🏷️ 16个游戏分类
- 🔍 实时搜索功能
- 📱 响应式设计
- ⌨️ 键盘快捷键

### 操作说明
- **鼠标点击** - 选择并启动游戏
- **方向键左/右** - 切换分类
- **方向键上/下** - 滚动游戏列表
- **键盘输入** - 搜索游戏
- **ESC** - 退出启动器
