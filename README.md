# 🎮 Python 小游戏合集

欢迎来到 Python 小游戏合集！这里包含了 **197个** 精美的 Python 小游戏，涵盖了25个分类和各种类型难度。

## 📁 文件说明

### 启动器
- **game_launcher.py** - 游戏启动器 v1.0（标准版）
- **game_launcher_v2.py** - 游戏启动器 v2.0（改进版，推荐使用）

## 🚀 快速开始

### 1. 安装依赖
```bash
pip install pygame
```

### 2. 使用游戏启动器（推荐）
```bash
python game_launcher_v2.py
```

### 3. 直接运行游戏
```bash
python simple_stardew_valley.py     # 简易星露谷
python multiplayer_bomberman.py     # 多人炸弹人
python life_simulator.py           # 生命模拟器
```

所有游戏都有独立的游戏体验和逻辑，无重复的"换皮"游戏：

#### 教育类游戏
- 🧪 化学元素猜猜乐：元素分类、属性记忆、多选答题
- 🚀 太空探索：飞船控制、行星发现、信息收集
- 🌍 环球冒险：地理知识、大洲探索、视觉反馈
- 🔢 数学大冒险：RPG升级、计时挑战、技能学习
- 🧬 细胞工厂：细胞器管理、蛋白质生产、升级系统
- ⏰ 时间旅行：历史知识、时代探索、时间碎片收集

#### 放置类游戏
- 🌾 农场大亨：多地块管理、作物多样、工人雇佣
- 🏭 生产线帝国：产品类型丰富、机器升级、质量控制
- 🏙️ 梦想城市：城市规划布局、人口增长、日夜循环
- ⛏️ 挖矿大亨：矿石稀有度、工具升级、存储出售
- ⚔️ 英雄传奇：职业多样、怪物战斗、队伍配合

#### 冒险解谜类游戏
- 📖 命运抉择：分支剧情、多重结局、物品收集
- 🔐 神秘逃脱：多房间探索、密码破解、道具组合
- 🔍 侦探事务所：线索收集、嫌疑人审讯、逻辑推理

#### 赛车类游戏
- 🏎️ 方程式赛车：赛道竞速、氮气加速、对手AI
- 🔄 漂移大师：手刹漂移、甩尾技巧、计分系统

## 📝 游戏开发说明

所有游戏均为独立Python文件，使用pygame库开发。代码包含详细中文注释，适合学习参考。

### 开发规范
- 所有游戏使用UTF-8编码
- 中文界面和注释
- 统一的错误处理
- 清晰的代码结构

### 音效系统
所有游戏内置音效系统，使用bytearray生成，无需外部音频文件。

## 🤝 贡献指南

欢迎提交新的游戏！

### 添加新游戏步骤
1. 在 `/workspace` 目录创建新的Python文件
2. 使用pygame库开发游戏
3. 添加中文界面和注释
4. 确保游戏有独特的玩法逻辑
5. 更新 `games_data` 字典到启动器
6. 更新本README文档
